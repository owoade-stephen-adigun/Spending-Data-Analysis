# Spending-Data-Analysis
Work designed to help a client analyze how he spent his money and how to properly  manage his money subsequently.

## Project Overview
This project analyzes my personal spending data for the month of September.
The goal is to understand spending patterns and identify areas for better
financial decision-making.

## Dataset
The dataset contains income (credit) and expenses (debit) categorized by item.

Columns:
- ITEMS: Description of transaction
- CREDIT: Income received
- DEBIT: Amount spent

## Objectives
- Clean and prepare the spending dataset
- Analyze income vs expenses
- Visualize spending patterns
- Draw insights for better budgeting

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Expenses outweigh income in some categories
- Food-related spending appears frequently
- Income sources are limited compared to expenses

## How to Run
1. Clone the repository
2. Install required libraries
3. Open the notebook and run all cells
